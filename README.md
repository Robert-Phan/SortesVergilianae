# Sortes Vergilianae

A simple, classical-themed recreation of the Sortes Vergilianae (drawing lots from Virgil’s Aeneid).

## Website

Visit robert-phan.github.

## Run locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Use

- Choose language: Latin or English (Dryden)
- Select books (1–12) to include
- Choose number of lines, then “Draw Your Fate”
- Use “lines above/below” to reveal context
- Click the title or subtitle for an explanation page
